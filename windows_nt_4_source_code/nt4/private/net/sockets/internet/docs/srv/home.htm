Thank you for using Microsoft Internet Explorer.

This home page is stored locally on your machine. To browse the Web pages on a Microsoft Internet Information Server installed on your local network, type the Windows Computer Name of that server computer in the Location box at the top of this browser. For example, if the server is named \\CorpNet, type:

CorpNet

To view the Web pages on a Microsoft Internet Information Server that is on the Internet, type the DNS name of the server in the Location box above. See the Installation and Planning Guide for more information.

A good place to begin browsing the Internet is www.microsoft.com, where you will find information about Microsoft products, and links to other useful sites. You can also get detailed information about Microsoft Internet Information Server including product documentation, updates, and additional samples. Another good place to start browsing is the Microsoft Network (MSN) Web site.
