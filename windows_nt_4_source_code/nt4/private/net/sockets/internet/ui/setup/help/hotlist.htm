Using the Hotlist Window

You can use the Hotlist window to maintain a list of frequently used files.

To add the current file to the Hotlist:

  • From the Navigate menu, choose Add Current to Hotlist.

    Or, click the Add Current to Hotlist toolbar button.

To open a file in the Hotlist:

  1. From the Navigate menu, choose Hotlist.

  2. Double-click the file you want to open (or select it and click Go To).

    Note - The Hotlist window remains open until you close it.

To delete a file from the Hotlist:

  1. From the Navigate menu, choose Hotlist.

  2. Select the file you want to delete and click the Delete button.

  3. Choose the OK button.

To copy or edit an entry in the Hotlist:

  1. From the Navigate menu, choose Hotlist.

  2. Select the file you want to copy or edit and click the Edit button.

  3. In the Edit Hotlist Entry dialog, you can type a different title or copy the URL.

  4. Choose the OK button.

To save the Hotlist as an HTML file:

  1. From the Navigate menu, choose Hotlist.

  2. Click Export.

  3. Type a filename. Use an .htm or .html extension.

  4. Choose the OK button.

To import a Hotlist:

  1. From the File menu, choose Open Local.

  2. Type or select the name of the Hotlist file you want to import and click OK.

    The hotlist is displayed as a list of links.

  3. Click a link you want to add to your Hotlist.

  4. When the file is displayed, choose Add Current to Hotlist from the Navigate menu.

  5. Click the Back button to return to the list of links.

  6. Repeat steps 3-5 for each link you want to add.
