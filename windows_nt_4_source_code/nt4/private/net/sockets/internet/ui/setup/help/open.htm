Opening a File

Clicking on a hyperlink is the easiest way to open a file. You can also use the Open URL or Open Local command to open a file if a hyperlink is not available and the file you want to open is not in your History or Hotlist window.

To open a file on the Internet:

  1. From the File menu, choose Open URL. Or, click the Open URL toolbar button.

  2. Type or paste the URL of the file you want to open. URLs are case sensitive and must not begin with a space.

  3. Choose the OK button.

    Note - URLs can also be entered in the URL field below the toolbar. Press Tab to select the URL field.

To open a file on your computer:

  1. From the File menu, choose Open Local.

  2. Type or select the filename.

  3. Choose the OK button.

If you have trouble opening a file, see troubleshooting for help.
