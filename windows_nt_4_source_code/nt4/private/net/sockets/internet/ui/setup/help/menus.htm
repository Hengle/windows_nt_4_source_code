File Menu

New Window
Opens a new window.

Open URL...
Opens a file from the Internet by entering a URL.

Open Local...
Opens a file from your computer or local network.

Close
Closes the current window. If only one window is open, this command quits Mosaic.

Save As...
Saves the current file to your computer.

Print
Prints the current file.

Page Setup...
Change the layout of printed pages.

Printer Setup
Change printer options.

Exit
Quits Mosaic.

Edit Menu

Cut
Removes the selected text and saves it to the clipboard.

Copy
Saves the selected text to the clipboard.

Paste
Pastes contents of the clipboard.

Select All
Selects all the text in the current document.

Find...
Searches a document for a specified word or phrase.

Find Again
Repeats the previous search.

View Source...
Displays the HTML tags in the current document.

Security
List installed security modules.

Preferences...
Change user options.

Helpers...
Configures helper applications.

Navigate Menu

Back
Displays the previous file.

Forward
Opposite of the Back command.

Home
Displays the home page.

History...
Lists the files loaded during the past 60 days. To return to a file, choose it from this list.

Hotlist...
Opens the Hotlist window.

Add Current to Hotlist
Adds the current file to the Hotlist window.

Load Missing Images
Loads missing images.

Reload
Reloads the current file.

Window Menu

Tile
Arranges open windows side by side so that all of them are visible.

Cascade
Arranges open windows so that the title bar of each window is visible.

The titles of all open windows are listed in the Window menu. To change to a different window, choose it from this list.

Help Menu

Mosaic Help Page
Displays help.

About Mosaic...
Displays program and system information.
