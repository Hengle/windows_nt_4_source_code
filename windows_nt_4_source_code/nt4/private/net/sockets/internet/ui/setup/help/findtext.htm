Searching a Document

You can use the Find command to search for a specific word or phrase in the current document. It does not allow you to search the Internet.

To search a document:

  1. From the Edit menu, choose Find.

  2. In the Text to Find field, type the text you want to find.

    To find text with specific capitalization, select the Match Case check box. To start searching from the top of the document, select the Start From Top check box.

  3. Click OK to start searching.

    If the text is found, it is highlighted.

  4. To find the next occurrence, choose Find Again from the Edit menu.
