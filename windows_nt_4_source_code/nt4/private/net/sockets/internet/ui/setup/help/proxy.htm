Setting up a Proxy Server

You can use a proxy server to access the Internet from behind a firewall.

To use a proxy server:

  1. From the Edit menu, choose Preferences.

  2. In the Proxy Server field, type the domain name and port number of the server used to bypass the gateway. See your system administrator for details.

  3. Choose the OK button.

To bypass a proxy server:

  1. From the Edit menu, choose Preferences.

  2. In the Bypass Proxy On field, type a comma-delimited list of systems, domains, and ports for which Mosaic should not use the proxy server.

    For example, "spyglass.com,:80" means to not proxy any accesses to machines inside the spyglass.com domain and not to proxy accesses on port 80 to any system.

    This field is ignored if no proxy server is specified in the Proxy Server field.

  3. Choose the OK button.
