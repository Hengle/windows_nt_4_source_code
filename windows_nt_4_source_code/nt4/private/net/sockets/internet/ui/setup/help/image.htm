Viewing Images

Mosaic can display GIF and JPEG images. If a helper application is configured to display these file types, it is used instead.

To display a GIF or JPEG image:

  • Click or enter the URL of an image file, such as this example of a JPEG file.

  • In a document window, choose Open Local from the File menu. Then type or select the filename of the image file.

To save an image file:

  • In the Image Viewer window, choose Save As from the File menu. Then type a filename. To save an image in another format, select a format from the Save File as Type list.

  • To save an image file (or any other file) directly to your disk without having Mosaic or a helper application display the image, press Ctrl and click the link that retrieves the image. Then choose the Save As button.

  • You can add the URL of an image to the Hotlist by choosing Add Current to Hotlist from the Navigate menu.

To copy an image file:

  • From the Edit menu, choose Copy to copy the image to the clipboard.

To close an image window:

  • From the File menu, choose Close.

    Choosing Exit quits the Mosaic program.
