Keyboard Commands

  • To stop a file transfer, press Esc or click the Stop button in the toolbar.

  • Press Ctrl while clicking a link to save the next file to your computer. Then choose the Save As button.

  • To load a missing image, click on the enclosed text with the right mouse button.

  • To save an inline image to your computer, press Ctrl and click on the image with your right mouse button.

  • Press Tab to select the URL field.

  • Press Shift while clicking a link to display the next file in a new window.

  • Press Ctrl-Tab to cycle between open document windows.
