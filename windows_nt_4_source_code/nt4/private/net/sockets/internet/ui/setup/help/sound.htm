Playing Sounds

Mosaic can play AU and AIFF sound files. If a helper application is configured to play these file types, it is used instead.

To play an AU or AIFF sound file:

  • Click or enter the URL of a sound file, such as this brief example.

  • In a document window, choose Open Local from the File menu. Then type or select the name of the sound file.

    Sound player controls:

     Start or resume playing the sound file.

     Stop playing and move to the beginning of the sound file.

     Pause at the current position.

    With your mouse or arrow keys, you can drag the scroll box to move to a different position in the sound file.

To play other sound formats, you must configure a helper application.

To save a sound file:

  • In the Sound window, choose Save As from the File menu. Then type a filename.

  • To save a sound file (or any other file) directly to your disk without having Mosaic or a helper application play the sound, press Control and click the link that retrieves the sound file. Then choose the Save As button.

  • You can also add the URL of a sound file to the Hotlist by choosing Add Current to Hotlist from the Navigate menu.

To close a Sound window:

  • From the File menu, choose Close.

    Choosing Exit quits the Mosaic program.
