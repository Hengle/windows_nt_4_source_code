Printing a File

If a printer is installed, you can print files displayed by Mosaic.

To print a file:

  1. From the File menu, choose Print.

    The Print dialog is displayed for the first document printed after starting Mosaic. Otherwise, printing begins immediately.

  2. If the Print dialog is displayed, set the printing options, as appropriate. Then, choose the OK button.

    Note - Before printing, Mosaic reformats the document to fit the current paper size.

To change printer settings:

  1. From the File menu, choose Print Setup.

  2. Set the printing options as appropriate. The available options depend on the printer installed.

  3. Choose the OK button.
