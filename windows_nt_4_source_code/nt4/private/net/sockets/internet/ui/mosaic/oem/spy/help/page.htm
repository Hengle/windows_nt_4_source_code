Changing Page Layout

You can change the margins, headers, and footers that Mosaic uses by default for printed pages.

To change the page layout:

  1. From the File menu, choose Page Setup.

  2. In the Page Margin fields, type the margin measurements in inches.

  3. In each Page Header and Page Footer field, type the appropriate variable to print the information you want to include.

    &w - window title
    &U - URL
    &d - date (Feb 24 1996)
    &D - date (24 Feb 1996)
    &t - time (05:30 PM)
    &T - time (17:30)
    &p - current page number
    &P - total number of pages
    && - single ampersand (&)

    You can combine variables with text. For example: Page &p of &P

  4. Choose the OK button.

To change the size of printed text:

  1. From the Edit menu, choose Preferences.

  2. Open the Style Sheet list, and then choose the style sheet you want to use.

  3. Choose the OK button.
