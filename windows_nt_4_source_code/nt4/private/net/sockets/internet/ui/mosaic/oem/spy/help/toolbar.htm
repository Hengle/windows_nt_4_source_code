Toolbar Buttons

Toolbar buttons offer quick access to frequently used menu commands.

  Home - Displays the home page.
  Open URL - Opens a file from the Internet by entering a URL.
  Open Local - Opens a file from your computer or local network.

  Reload - Reloads the current file.
  Load Missing Images - Loads missing images.

  Find - Searches a document for a specified word or phrase.
  Find Again - Repeats the previous search.

  Print - Prints the current file.
  Save As - Saves the current file to your computer.

  Hotlist - Opens Hotlist window.
  Add Current to Hotlist - Adds the current file to the Hotlist window.

  Help - Displays help.
  Stop - Cancels file transfer.

To display the toolbar:

  1. From the Edit menu, choose Preferences.

  2. Select the Show Toolbar check box.

  3. Choose the OK button.
