5/31/96 - NoelC

This is the non-redistributable source share for Creative Labs drivers.
For a full log of changes, please reference \ntos\dd\sound\sndblst.

The directory structure is as follows:

creative\		- contains all kernel mode Creative Lab drivers.
creative\sndblst 	- contains the kernel mode SB16/SBPro/SB driver.
creative\sbpnp		- contains the kernel mode SBPnP DRVLIB driver.
